` tags added after every line: **API de Transport en Tunisie Documentation** Bienvenue dans la documentation de l'API de Transport en Tunisie.
Cette API fournit des données et des fonctionnalités pour aider les développeurs à intégrer des informations de transport public dans leurs applications.
Que vous construisiez une application mobile, un site web ou tout autre service nécessitant des données de transport en Tunisie, cette API vous fournira les outils nécessaires pour accéder à ces informations de manière fiable et efficace.
--- ### Endpoints Disponibles 1. **/stations**
- Méthode : GET
- Description : Récupère la liste des stations de transport disponibles en Tunisie.
- Paramètres :
- `latitude` (facultatif) : Latitude de la zone pour laquelle vous souhaitez récupérer les stations.
- `longitude` (facultatif) : Longitude de la zone pour laquelle vous souhaitez récupérer les stations.
2. **/lignes**
- Méthode : GET
- Description : Récupère la liste des lignes de transport disponibles en Tunisie.
- Paramètres :
- Aucun paramètre requis.
3. **/horaires**
- Méthode : GET
- Description : Récupère les horaires des prochains passages pour une station spécifique.
- Paramètres :
- `station_id` (obligatoire) : ID de la station pour laquelle vous souhaitez récupérer les horaires.
- `date` (facultatif) : Date pour laquelle vous souhaitez récupérer les horaires. Format : AAAA-MM-JJ.
4. **/tarifs**
- Méthode : GET
- Description : Récupère les tarifs des différents modes de transport en Tunisie.
- Paramètres :
- Aucun paramètre requis.
--- ### Exemple d'Utilisation **Récupérer les stations les plus proches :**
``` GET /stations?latitude=36.8065&longitude=10.1815 ``` **Récupérer les horaires pour une station spécifique :**
``` GET /horaires?station_id=123&date=2024-02-08 ``` --- ### Réponses de l'API L'API répondra avec des données au format JSON qui incluent les informations demandées.
Les erreurs seront renvoyées avec un code d'erreur approprié ainsi qu'un message descriptif pour aider à diagnostiquer les problèmes.
--- ### Authentification Actuellement, cette API ne nécessite pas d'authentification.
Cependant, il est fortement recommandé de limiter l'accès à l'API en utilisant des méthodes d'authentification appropriées pour assurer la sécurité des données et éviter toute utilisation abusive.
--- Merci d'utiliser l'API de Transport en Tunisie.
Si vous avez des questions ou des commentaires, n'hésitez pas à nous contacter à l'adresse support@transport-tunisie-api.com.
